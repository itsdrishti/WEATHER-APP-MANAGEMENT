# WEATHER-APP-MANAGEMENT
Developed a responsive Weather Forecast Dashboard using HTML, CSS, JavaScript, and OpenWeatherMap API to display real-time weather updates, 5-day forecasts, humidity, wind speed, and temperature with dynamic UI, API integration, and responsive design.
Smart Weather Forecast Dashboard
A responsive Weather Forecast Dashboard built using HTML, CSS, JavaScript, and OpenWeatherMap API. The project provides real-time weather updates, 5-day forecasts, temperature, humidity, wind speed, and dynamic weather conditions with a modern user interface.

Features
Real-time weather updates

5-day weather forecast

Search weather by city

Temperature, humidity & wind speed

Responsive design

Dark/Light mode

Dynamic weather icons

Error handling for invalid cities
